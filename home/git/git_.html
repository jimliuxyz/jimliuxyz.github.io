SVN(subversion) 與 Git

cvs

  • 集中式

svn

  • 集中式:
  • 所有版本都在svn server上,client端僅存有某一特定版本,所以版本切換時很慢
  • server一掛就全部死光.
  • 紀錄diff patch,每個版本透過patch還原,所以換版本時也很慢.

git

  • 分散式
  • 本地端含有所有版本,遠端僅進行與各個用戶端同步. 紀錄個別檔案
